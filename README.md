# One Golden Mile Properties Website

## Overview
This project is a single-page website for One Golden Mile Properties, a channel partner business that connects buyers and sellers in the real estate market. The website serves as a landing page showcasing the services offered, property categories, and additional support for clients.

## Project Structure
The project consists of the following files:

- **index.html**: The main HTML document that contains the structure of the landing page.
- **css/styles.css**: The CSS file that defines the visual style of the website, ensuring a clean, minimal, and professional look.
- **js/script.js**: The JavaScript file intended for future functionality, including interactions and dynamic elements.

## Features
- Responsive design for mobile-first experience
- Clean and professional layout with light background and subtle gold accents
- Sections for business overview, property categories, service locations, approvals, remote support services, and contact details
- Placeholder for a future customer support chatbot

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. Modify the `css/styles.css` file to customize styles as needed.
4. Add any JavaScript functionality in the `js/script.js` file.
